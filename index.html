🧪 Atelier IA : Test de Prompts

📝 Mode d'emploi

  1. Choisissez un mauvais ou un bon prompt parmi les exemples ci-dessous
  2. Cliquez sur "📋 Copier ce prompt"
  3. Cliquez sur "🚀 Tester sur ChatGPT" ou "Claude"
  4. Collez le prompt (appui long puis "Coller" sur mobile)
  5. Comparez les résultats ! 🎯

📝 Exemple 1 : Lettre de motivation

❌ MAUVAIS PROMPT
Peux-tu me faire une lettre de motivation ?
🚀 Tester sur ChatGPT 🚀 Tester sur Claude
✅ BON PROMPT
Tu es conseiller RH, fais-moi une lettre de motivation pour le poste de jardinier, voici mon CV et voici l'offre de poste, adapte-la en fonction de mon profil et fais-moi le contenu sur une page, un ton naturel, pas trop formel, (que l'on ne distingue pas que c'est rédigé par une IA).
